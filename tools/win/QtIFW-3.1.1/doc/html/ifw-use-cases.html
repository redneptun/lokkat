End User Workflows

The end user experience is similar for offline and online installers. Along with your application, the installers install a maintenance tool that consists of a package manager, an updater, and an uninstaller. End users can use the maintenance tool to add, update, and remove components. The maintenance tool connects to an external repository to fetch the components to add or update. You can specify the repository in the configuration file or end users can specify it in the maintenance tool settings.

You can support the following end user workflows:

  • Initial Installation
  • Adding Components
  • Removing Components
  • Updating Components
  • Specifying Settings
