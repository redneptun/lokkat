Updating Components

The following image illustrates the default workflow for updating installed components:

This section uses the Qt 5 Maintenance Tool run on OS X as an example implementation of how end users can update installed components.

Starting Updater

When end users start the Maintenance Tool, the introduction page opens:

When end users select Update components, and then Continue, the component selection page opens.

Selecting Components to Update

The updater displays a list of available updates that end users can select from.

When end users select Continue, the ready to update page opens.

Updating Selected Components

The ready to update page informs end users that the components are updated when they select Update.

The update page displays information about how updating is progressing. End users can select Show Details to view more information.

When update is complete, the update finished page opens.
